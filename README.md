# HourXchange 프로젝트

HourXchange는 시간 기반 크레딧 거래 플랫폼으로, 사용자가 서비스 제공 및 요청을 할 수 있는 환경을 제공합니다.  
이 프로젝트는 프론트엔드와 백엔드로 구성되며,  
백엔드는 **Spring Boot**와 **Flask API**(LLM)를 사용하고,  
프론트엔드는 **React**와 **Vite**로 개발되었습니다.  
또한, 검색 엔진으로는 **Elastic Search**가 사용됩니다.
